INSTR_* Instructions

Instruction Operation Code (Value) Description
CPU Execute INSTR_CPU_EXEC (0) Executes the operation specified in REG_OP using the provided registers and stores the result.
Copy INSTR_COPY_FROM_TO_ADDRESS (1) Copies data from a source address to a destination address.
Read Memory INSTR_READ_FROM_ADDRESS (2) Reads data from a specified memory address and stores it in REG_RES.
Jump INSTR_JUMP (3) Jumps unconditionally to the address provided as an argument.
Conditional Jump INSTR_JUMP_IF (4) Jumps to the address provided as an argument if REG_BOOL_RES contains 1.
Conditional Jump INSTR_JUMP_IF_NOT (5) Jumps to the address provided as an argument if REG_BOOL_RES contains 0.
Conditional Jump INSTR_JUMP_ERR (6) Jumps to the address provided as an argument if REG_ERROR is not empty.

OP_* Instructions

Instruction Operation Code (Value) Registers Description
Add OP_ADD (0) REG_A, REG_B, REG_RES Adds values in REG_A and REG_B, stores the result in REG_RES.
Subtract OP_SUB (1) REG_A, REG_B, REG_RES Subtracts REG_B from REG_A, stores the result in REG_RES.
Increment OP_INCR (2) REG_A, REG_RES Increments the value in REG_A, stores the result in REG_RES.
Decrement OP_DECR (3) REG_A, REG_RES Decrements the value in REG_A, stores the result in REG_RES.
Divide OP_DIV (4) REG_A, REG_B, REG_RES Divides REG_A by REG_B, stores the result in REG_RES.
Modulus OP_MOD (5) REG_A, REG_B, REG_RES Computes the modulus of REG_A by REG_B, stores the result in REG_RES.
Multiply OP_MUL (6) REG_A, REG_B, REG_RES Multiplies REG_A by REG_B, stores the result in REG_RES.
Is Number OP_IS_NUM (7) REG_A, REG_BOOL_RES Checks if REG_A is a number, stores the result in REG_BOOL_RES.
Equal OP_CMP_EQ (8) REG_A, REG_B, REG_BOOL_RES Compares if REG_A equals REG_B, stores the result in REG_BOOL_RES.
Not Equal OP_CMP_NEQ (9) REG_A, REG_B, REG_BOOL_RES Compares if REG_A is not equal to REG_B, stores the result in REG_BOOL_RES.
Less Than OP_CMP_LT (10) REG_A, REG_B, REG_BOOL_RES Checks if REG_A is less than REG_B, stores the result in REG_BOOL_RES.
Less Than or Equal OP_CMP_LE (11) REG_A, REG_B, REG_BOOL_RES Checks if REG_A is less than or equal to REG_B, stores the result in REG_BOOL_RES.
Contains OP_CONTAINS (12) REG_A, REG_B, REG_BOOL_RES Checks if REG_A contains REG_B, stores the result in REG_BOOL_RES.
Get Length OP_GET_LENGTH (13) REG_A, REG_RES Gets the length of the value in REG_A, stores the result in REG_RES.
Starts With OP_STARTS_WITH (14) REG_A, REG_B, REG_BOOL_RES, REG_RES Checks if REG_A starts with REG_B, stores the result in REG_BOOL_RES and the tail of the string will be in REG_RES..
Get Column OP_GET_COLUMN (15) REG_A, REG_B, REG_C, REG_RES Extracts a column from a string, specified by the delimiter in REG_C and the column number in REG_B.
Replace Column OP_REPLACE_COLUMN (16) REG_A, REG_B, REG_C, REG_D, REG_RES Replaces a specific column in a string with a new value in REG_D.
Concat with OP_CONCAT_WITH (17) REG_A, REG_B, REG_C, REG_RES Concatenates two strings with a delimiter, stores the result in REG_RES.
Read Input OP_READ_INPUT (18) REG_A, KEYBOARD_BUFFER Reads input from the keyboard, stores it in KEYBOARD_BUFFER. Use REG_A to set read mode: KEYBOARD_READ_LINE, KEYBOARD_READ_LINE_SILENTLY, KEYBOARD_READ_CHAR, KEYBOARD_READ_CHAR_SILENTLY
Display OP_DISPLAY (19) DISPLAY_BUFFER, DISPLAY_COLOR Displays a string without a newline, using the color specified.
Display with Newline OP_DISPLAY_LN (20) DISPLAY_BUFFER, DISPLAY_COLOR Displays a string followed by a newline, using the color specified.
Read Block OP_READ_BLOCK (21) REG_A, REG_B, REG_RES, REG_ERROR Reads a block of data from a disk, specified by the disk name in REG_A and block number in REG_B. In case of any errors REG_ERROR will be set.
Write Block OP_WRITE_BLOCK (22) REG_A, REG_B, REG_C, REG_BOOL_RES, REG_ERROR Writes a string to a disk block. In case of any errors REG_ERROR will be set.
Set background color OP_SET_BACKGROUND_COLOR (23) DISPLAY_BACKGROUND Changes the background color based on the state of DISPLAY_BACKGROUND register which should contain a COLOR_* constant.
Render bitmap OP_RENDER_BITMAP (24) REG_A, REG_B Allows to print colored cells on the screen based on the provided bitmap. Set address of the first bitmap line to REG_A and the address after the last line to REG_B. Bitmap lines should contain only letters B(black), g(green), y(yellow), r(red), b(blue), m(magenta), c(cyan), w(white), o(orange), n(no color)
No Operation OP_NOP (29) REG_A Performs no operation, used for delays or placeholders. Set delay value in seconds(for example, 0.5) to REG_A.
Halt OP_HALT (30) None Halts the CPU and stops the system.
Unknown OP_UNKNOWN (31) None Resets the operation code for safety after execution.
